Direct Authentication Example

This page demonstrates how to use direct authentication between Open WebUI and Canvas Editor

How It Works

This example shows how to directly transfer a user from Open WebUI to Canvas Editor without requiring an HTML authentication page.

Note: For this example, we're using a mock JWT token. In a real application, you would use a valid JWT token from your authentication system.

1. Include the Editor Button Script

<script type="module" src="/integration/editor-button-direct.js"></script>

2. Create a Container for the Button

<div id="editor-button-container"></div>
